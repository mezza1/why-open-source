# Why You Should Use Open Source Models

A single-page site listing reasons to build on open-weight AI models instead of closed ones — each reason expands for detail and sources, and contributions are welcome.

**Live:** https://mezza1.github.io/why-open-source/

Plain static HTML/CSS, no build. Edit reasons in `index.html` (one `<li>` each; numbering is automatic). Run locally with `python3 -m http.server 4173`. Hosted on GitHub Pages — pushes to `main` auto-deploy.

To contribute, open a PR or issue with a new reason or a better source.

Collated by [Doubleword](https://doubleword.ai), who ❤️ open source.
